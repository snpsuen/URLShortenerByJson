Request/response API

Request header and payload POST /newurl HTTP/1.1
Content-type: application/json

{ "url": "https://www.google.com" }

Command line example using curl: 

curl -X POST -H "Content-type: application/json" -d "{\"url\" : \"https://www.google.com\"}" http://shortenurl.org/newurl

Response header and payload 

HTTP/1.1 200 OK
Content-Type: application/json

{ "url": "https://www.google.com", "shortenUrl": "https://shortenurl.org/g20hi3k9t"}
